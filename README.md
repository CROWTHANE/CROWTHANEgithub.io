# Retro Arcade

A small static website that includes three retro browser games: Pong, Snake, and Breakout.

How to run
1. Put the files in a folder (index.html, styles.css, main.js).
2. Open index.html in a modern browser (Chrome, Firefox, Edge).
3. Play! High scores are saved to localStorage.

Controls
- Pong: W/S or Up/Down. Pause: P.
- Snake: Arrow keys or WASD. Pause: P.
- Breakout: Mouse to move or Left/Right arrows. Pause: P.

You can host this on GitHub Pages or any static host.
